Markdown 批量清洗

指定顶层目录后扫描其中的 Markdown 文件，支持多选并按同一套正则规则批量清洗。

可填写绝对路径，也可填写相对路径；相对路径会按当前项目目录解析。

默认规则会直接显示在下方，可按需启用、禁用或编辑。
